Whac‑a‑mole

Класична аркадна гра "Whac-a-Mole", реалізована на React.
Гравець обирає рівень складності та розмір поля, після чого намагається влучити по кротах, які з'являються у випадкових клітинках. Результат підраховується та відображається після завершення гри.

Можливості

  • Три рівні складності: легкий, важкий, неможливий
  • Два розміри поля: 3×3 та 5×5
  • Збереження налаштувань у localStorage
  • Анімований виліт крота з нори
  • Таймер 30 секунд

Інструкція по використанню

  1. Клонування репозиторію
    git clone https://github.com/neizvestniy01/whac-a-mole.git
    cd whac-a-mole
  2. Встановлення залежностей
    npm install
  3. Запуск
    npm start
